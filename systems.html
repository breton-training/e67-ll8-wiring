Verification Sequence — Before Applying Power
Applies to both factory pigtail and new connector. Same procedure, same expected values.
Step 1 ECM disconnected from harness Measure OBDII Pin 6 to Pin 14 Expect: ~120Ω (terminator only)
Step 2 ECM connected · ignition off Measure OBDII Pin 6 to Pin 14 Expect: ~60Ω (internal + external in parallel)
Note If Step 2 reads ~120Ω instead of ~60Ω — ECM has no internal terminator. External terminator is the only one. CAN bus will still function — this is not a failure condition, just a finding to document.
STOP If Step 1 reads open (OL) — no terminator present. Do not proceed. Add external 120Ω before continuing. If readings are not as expected — investigate CAN wiring before applying power.

Ignition Coils (COP)
6 coil-on-plug · 12V from PWR/TRN relay via INJ B (Fuse 56, 20A) · ECM controls dwell/timing per coil
▾
Coil Pin A = 12V (circuit 1339 PK, shared bus J102 from INJ B Fuse 56 20A). Coil Pin B = ECM ignition control signal (IC). Coil Pin C = ground (circuit 550 BK to G109). ECM varies dwell and timing on Pin B — not a simple ground pull like injectors.

VVT · CMP · CKP
Exhaust cam VVT only (not intake) · Used for internal EGR · ECM inputs: CKP, CMP, EOP, ECT, load
▾
LL8 VVT is exhaust cam only — confirmed. Delays exhaust valve closing to draw exhaust gas back into cylinder (internal EGR). Intake cam is fixed. The camshaft position actuator solenoid is hydraulically operated via engine oil pressure — EOP sensor input is a prerequisite for VVT operation.

Open Anomalies & Verification Items
Unresolved items from handoff — do not wire without verification
▾

Anomaly 1 — IC 4 Dual Pin (UNRESOLVED)

X2-18 (D-GN/WH, 0.5mm², circuit 2124) and X2-34 (D-GN/WH, 0.35mm², circuit 2124) both labeled IC 4 Control with same circuit number. Factory coil diagram shows X2-34 as active coil 4 control. Do not connect both pins to coil 4. Verify against factory diagram before final wiring.

Anomaly 2 — Starter Enable Dual Pin (RESOLVED)

X1-56 (YE/BK, 625) and X2-43 (YE, 6259) appeared as duplicate "Starter Enable Relay Control." Factory starting system diagram confirms: X2-8 (circuit 1433 D-GN) = starter relay coil control for 4.2L. X1-50 (circuit 806 PU) = direct starter solenoid enable output. X1-56 (circuit 625 YE/BK) = starter relay coil in starting diagram. Verify X1-56 vs X2-8 function before final wiring.

Open Item — Internal CAN Terminator Verification

Forum-reported that E67 has internal 120Ω CAN terminator. Verify empirically: with ECM disconnected, measure X3-33 to X3-53 — should read open if no internal terminator. With ECM connected, ignition off — should read ~60Ω if internal terminator present (two 120Ω in parallel with external). If reads 120Ω with ECM connected, no internal terminator and external 120Ω is the only terminator.

Open Item — Fan Control Calibration Status

Factory diagram confirms PWM fan output pins exist on 4.2L ECM (X1-23, X2-21, X1-4, X1-51). Whether the YRH broadcast code calibration has these outputs active or enabled is not confirmed. Verify in HP Tuners VCM Editor after bench flash before planning electric fan install.
